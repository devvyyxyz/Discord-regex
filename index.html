Discord AutoMod Regex Showcase

Explore regex rules to enhance your Discord server's auto moderation.

Spam Protection

This rule detects repeated characters in a message:

/(.)\1{4,}/g

Example: Matches “heyyyyyyy” or “woooowww”.

Profanity Filter

This rule blocks offensive words:

/(badword|offensiveword)/gi

Example: Matches common offensive terms.

Link Detection

This rule detects URLs and prevents users from posting links:

https?:\/\/(www\.)?[-a-zA-Z0-9@:%._\+~#=]{1,256}\.[a-zA-Z0-9()]{1,6}\b([-a-zA-Z0-9()@:%_\+.~#?&//=]*)

Example: Matches “http://example.com” or “https://discord.gg/abcdef”.

Mention Spam Detection

This rule blocks users from spamming mentions:

/@\w+/g

Example: Matches “@everyone” or “@username”.

Caps Lock Detection

This rule detects if a message is written in all capital letters:

^[A-Z\s]+$/g

Example: Matches “HELLO THERE” or “THIS IS SPARTA”.

Discord Invite Link Detection

This rule detects Discord invite links in messages:

https?:\/\/(www\.)?(discord\.(gg|com)\/invite|discordapp\.com\/invite)\/[a-zA-Z0-9]{1,}/g

Example: Matches “https://discord.gg/abcdef” or “discord.com/invite/xyz123”.
